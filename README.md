#  Python Internship Projects - CodeAlpha

This repository contains my Python internship projects completed during the CodeAlpha Internship.   

###  Task 1: Hangman Game
A simple text-based word guessing game.
- Uses random word selection
- Player guesses letters one by one
- Maximum 6 wrong attempts allowed
- Concepts used:
  - Loops
  - Conditions
  - Lists
  - Strings
  - Random module


### Task 2: Stock Portfolio Tracker
A Python program to calculate total investment value.
- Takes stock name and quantity from user
- Uses dictionary for stock prices
- Calculates total investment
- Saves result into a text file
- Concepts used:
  - Dictionary
  - Input/Output
  - File Handling


###  Task 3: Basic Chatbot
A simple rule-based chatbot.
- Takes user input
- Gives predefined responses
- Runs until user types "bye"
- Concepts used:
  - Functions
  - if-elif conditions
  - Loops
